# MarketX Marketplace Project

## Project Setup
To set up the MarketX Marketplace project, follow these steps:
1. Clone the repository:
   ```bash
   git clone https://github.com/santero990/marketstore.git
   cd marketstore
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Configure environment variables.

## Features
- User-friendly interface for both buyers and sellers.
- Secure payment processing.
- Real-time notifications.
- Comprehensive search and filter options.

## Usage Instructions
1. Start the development server:
   ```bash
   npm start
   ```
2. Open your browser and navigate to `http://localhost:3000` to access the marketplace.
3. Register or log in to start using the features.
